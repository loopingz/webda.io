# @webda Framework Tutorials

Welcome to the @webda framework tutorials section! Here you'll find step-by-step guides to help you build various applications using @webda. Each tutorial is accompanied by a sample project located in the `samples/` directory of this repository.

## Available Tutorials

### 1. Building a Webmail Application
   - **Description:** Learn how to create a basic webmail application, including managing users, emails, and folders.
   - **Tutorial:** [Webmail Tutorial](./Webmail.md)
   - **Sample Project:** `samples/webmail/` (Link will be to the directory on the repository browser)

### 2. Building a CRM (Customer Relationship Management) System
   - **Description:** This tutorial guides you through building a simple CRM to manage contacts, companies, and sales opportunities.
   - **Tutorial:** [CRM Tutorial](./CRM.md)
   - **Sample Project:** `samples/crm/`

### 3. Building a CMS (Content Management System)
   - **Description:** Discover how to construct a basic CMS for creating and managing content like articles and pages.
   - **Tutorial:** [CMS Tutorial](./CMS.md)
   - **Sample Project:** `samples/cms/`

We encourage you to go through these tutorials to get a practical understanding of the @webda framework's capabilities.
